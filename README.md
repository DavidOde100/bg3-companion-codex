# WEB103 Project 1 - *Tavern Codex: A Baldur's Gate 3 Companion Guide*

Submitted by: **David Odejimi**

About this web app: **The web app will be a Baldur’s Gate-themed character codex that displays unique characters as fantasy-style cards. Each card will show the character’s name, race, class, role, and short description. When a user clicks a card, they will be taken to a detailed page with all character information from the database.**

Time spent: **4.5** hours

## Notes

One of the biggest challenges I encountered while building this project was deciding on an idea that I was genuinely interested in. At first I went with Cosmic Wonder with galaxy and stuff which I built but I didn't like it because it was cool enough so I thought of a game I like to play which was Baldur's Gate. Once I settled on creating a Baldur’s Gate 3 Companion Codex, the rest of the project became much easier to plan and develop because I was familiar with the theme and characters. Another challenge was designing the CSS to capture the visual style of Baldur’s Gate, including the dark fantasy aesthetic, gold accents, and immersive layout. I also anticipate that obtaining and managing larger amounts of character data in future iterations may be more challenging, but I am interested in exploring that further in the next class.
